# Restaurant Order System

A basic restaurant order and bill calculation console application developed with C#.

## Features
- Food, drink and dessert menu
- User-based order selection
- Total bill calculation
- Console-based menu system
- Reorder and exit options

## Technologies Used
- C#
- .NET
- Console Application

## Project Purpose
This project was developed to improve my programming logic and gain experience with conditional statements, loops and menu-based console applications in C#.
